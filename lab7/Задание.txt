Часть 1. Сериализация

1. Изучите класс Tank, для которого предусмотрена сериализация в XML.
	Что вы видите необычного? Запишите ответ в отчет по работе.

2. Создайте класс "ангар" (Hangar), содержащий список танков. Снабдите его атрибутом
    [XmlRootAttribute()]
	, указывающим, что класс может выступать в качестве корневого узла XML-документа.

	Список танков снабдите атрибутом 
	[XmlArrayAttribute("Tanks")]
	, указывающим, что его следует сериализовать в элемент-коллекцию

3.	В консольном приложении напишите код, создающий ангар со следующими танками:

Т-34
Масса 28 т. 
Скорость 56 км/ч
Лошадиных сил 480 
Калибр орудия 57 мм

Иосиф Сталин 
Масса 45 т. 
Скорость 34 км/ч
Лошадиных сил 600 
Калибр орудия 122 мм

Panther 
Масса 44.45 т. 
Скорость 55 км/ч
Лошадиных сил 650 
Калибр орудия 75 мм

Tiger
Масса 55.5 т. 
Скорость 40 км/ч
Лошадиных сил 650 
Калибр орудия 88 мм

Можете также добавить свои танки, если хотите.

4. Создайте объект класса XMLSerializer при помощи следующего кода
var s = new XmlSerializer(typeof(Hangar));
Этот класс служит для вывода сериализуемых объектов в поток (Stream);

5. Создайте объект класса XmlWriter при помощи следующего кода
var w =  XmlWriter.Create("MyTanks.xml");
Класс XmlWriter - абстрактный, а его метод Create предоставляет конкретную реализацию для вывода потока в xml-файл

6. Используйте метод Serialize созданного объекта XmlSerializer, чтобы выгрузить ваши танки в XML-файл
Не забудьте закрыть файл за собой: 
w.Close();

7. Выполните программу и убедитесь, что файл с танками сохранился на диске. Сделайте скриншот файла и вставьте в отчет.


Часть 2. Десериализация
1. Закомментируйте код, создающий танки и сохраняющий их в файл. Оставьте только создание XmlSerializer - он нам еще пригодится.

2. Создайте объект класса XmlReader при помощи следующего кода
var w =  XmlReader.Create("MyTanks.xml");
Класс XmlReader - абстрактный, а его метод Create предоставляет конкретную реализацию для чтения в поток из xml-файла

3. Используйте метод Deserialize созданного объекта XmlSerializer, чтобы загрузить ваши танки из XML-файла

4. Выведите все танки ангара в консоль для проверки (имена и характеристики). Используйте перегрузку метода ToString()
 
5. Сделайте скриншот консоли с танками и вставьте в отчет.

6. Вставьте в отчет листинг вашей программы и классов - и можете сдавать работу!
